UClean 清洁度分析 Windows 便携版
================================

使用方法
1. 将压缩包完整解压到一个普通文件夹。
2. 双击“UClean清洁度分析.exe”。
3. 选择标准模型、未染色模型、刷牙前模型和刷牙后模型。
4. 选择分割区域后执行清洁度计算。

批量处理
1. 在第一页把“处理模式”切换为“文件夹批量处理”。
2. 选择包含多个刷牙后模型的文件夹（支持 PLY / OBJ / STL）。
3. 如需阈值，勾选后会进入第二页；确认阈值后自动返回第一页。
4. 所有模型共用一次确认的颜色空间和阈值。
5. 每个模型的结果按原文件名保存，批量汇总表为 batch_cleanliness_summary.csv。

注意事项
- 无需安装 Python、Anaconda 或其他运行环境。
- 不要只把 EXE 单独复制出来，_internal 文件夹是程序运行所必需的。
- 默认分割模板和示例牙模已经包含在 _internal 文件夹中。
- 计算结果默认写入程序旁边的 selectable_cleanliness_results 文件夹。
- 首次启动或加载大型牙模时可能需要稍等。
- 精简便携版不包含 GPU 自动单牙分割模块。
- 仅支持 64 位 Windows 10/11。
